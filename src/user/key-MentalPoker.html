1. Поиск взаимно простых чисел

Предварительный этап необходим для выбора параметров протокола. Участники выбирают несекретное большое простое число p
p = ;

Затем Алиса выбирает случайно число cA, взаимно простое с p-1. Независимо и аналогично Боб выбирает случайно число cB, взаимно простое с p-1.

2. Поиск пары чисел

Теперь Алиса и Боб должны вычислить dA и dB по обобщенному алгоритму Евклида.

Алиса: cA dA mod (p - 1) = 1;
Боб: cB dB mod (p - 1) = 1;

Исходные данные:
cA = ;
cB = ;
p = ;

Алиса держит в секрете числа cA и dA. Боб держит в секрете числа cB и dB

3. Выбор случайных карт

Алиса выбирает случайно 3 различных числа alpha, beta, gamma в промежутке от 1 до p-1 в открытом виде и передает их Бобу.

4. Раздача карт - шаг 1

Алиса вычисляет числа:
u1 = alpha cA mod p;
u2 = beta cA mod p;
u3 = gamma cA mod p;

И высылает их Бобу, предварительно перемешав случайным образом.

Исходные данные:
alpha = ;
beta = ;
gamma = ;
cA = ;
p = ;

5. Раздача карт - шаг 2

Боб получает 3 числа и выбирает случайно одно из них, например u2 и отправляет его Алисе. Это и будет карта, которая достанется ей в процессе раздачи. Алиса, получив сообщение может вычислить какой карте соответствует принятое от Боба число:
ualice = u2dA mod p;

Исходные данные:
u2 = ;
dA = ;
p = ;

6. Раздача карт - шаг 3

Боб продолжает свои действия, он вычисляет для оставшихся двух чисел:
v1 = u1cB mod p;
v3 = u3cB mod p;

Исходные данные:
u1 = ;
u3 = ;
cB = ;
p = ;

С вероятностью 1/2 он переставляет эти два числа и отправляет Алисе.

7. Раздача карт - шаг 4

Алиса выбирает случайно одно из полученных чисел, например v1 и вычисляет число:
w1 = v1dA mod p;

И отправляет это число обратно Бобу. Боб вычисляет число:
z = w1dB mod p;

И узнает свою карту. Оставшаяся карта остается в прикупе.

Исходные данные:
v1 = ;
dA = ;
dB = ;
p = ;
