1. Поиск взаимно простых чисел

Предварительный этап необходим для выбора параметров протокола. Участники выбирают несекретное большое простое число p
p = ;

Затем Алиса выбирает случайно число cA, взаимно простое с p-1. Независимо и аналогично Боб выбирает случайно число cB, взаимно простое с p-1.

2. Поиск пары чисел

Теперь Алиса и Боб должны вычислить dA и dB по обобщенному алгоритму Евклида.

Алиса: cA dA mod (p - 1) = 1;
Боб: cB dB mod (p - 1) = 1;

Исходные данные:
cA = ;
cB = ;
p = ;

Алиса держит в секрете числа cA и dA. Боб держит в секрете числа cB и dB

3. Раздача карт - шаг 1

Алиса вычисляет числа:
u1 = alpha cA mod p;
u2 = beta cA mod p;
u3 = gamma cA mod p;

Исходные данные:
alpha = ;
beta = ;
gamma = ;
cA = ;
p = ;

4. Отправка Бобу - шаг 2

Алиса пермешивает числа, которые получила на предыдущем шаге и отправляет их Бобу.

Правило перемешивания - 123 -> 231

Исходные данные:
u1 = ;
u2 = ;
u3 = ;

5. Раздача карт - шаг 3

Боб получает 3 числа и выбирает случайно одно из них, например первое и отправляет его Алисе. Это и будет карта, которая достанется ей в процессе раздачи. Алиса, получив сообщение может вычислить какой карте соответствует принятое от Боба число:
ualice = ubob1dA mod p;

Исходные данные:
ubob1 = ;
dA = ;
p = ;

6. Раздача карт - шаг 4

Боб продолжает свои действия, он вычисляет для оставшихся двух чисел:
v1 = u1cB mod p;
v3 = u3cB mod p;

Исходные данные:
u1 = ;
u3 = ;
cB = ;
p = ;

7. Отправка Алисе - шаг 5

Боб с вероятностью 1/2 переставляет два числа, которые получил на предыдущем шаге и отправляет их Алисе.

Правило перемешивания - 12 -> 21

Исходные данные:
v1 = ;
v3 = ;

8. Раздача карт - шаг 6

Алиса выбирает случайно одно из полученных чисел, например второе и вычисляет число:
w = valice2dA mod p;

И отправляет это число обратно Бобу. Боб вычисляет число:
z = wdB mod p;

И узнает свою карту. Оставшаяся карта остается в прикупе.

Исходные данные:
valice2 = ;
dA = ;
dB = ;
p = ;
